# GCP Virtual Private Cloud (VPC)

In **Google Cloud Platform (GCP)**, a **Virtual Private Cloud (VPC)** is a **secure, isolated network** where you deploy and manage your cloud resources such as virtual machines, databases, and load balancers.

It acts like your **private data center network**, but fully managed in the cloud.
