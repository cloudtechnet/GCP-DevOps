## gcloud CLI – Basic Commands

The **gcloud CLI** is the primary command-line tool used to interact with **Google Cloud Platform (GCP)**. These basic commands are essential for **daily cloud administration, DevOps tasks, and interview preparation**.

---

## 1️⃣ Authentication Commands

### Login to GCP

```bash
gcloud auth login
```

* Authenticates your Google account
* Opens browser for secure login

### List authenticated accounts

```bash
gcloud auth list
```

---

## 2️⃣ Configuration Commands

### View current configuration

```bash
gcloud config list
```

### Set active project

```bash
gcloud config set project PROJECT_ID
```

### Set default region

```bash
gcloud config set compute/region REGION_NAME
```

### Set default zone

```bash
gcloud config set compute/zone ZONE_NAME
```

---

## 3️⃣ Project Management Commands

### List projects

```bash
gcloud projects list
```

### Describe a project

```bash
gcloud projects describe PROJECT_ID
```

---

## 4️⃣ Compute Engine Commands (VM Basics)

### List virtual machines

```bash
gcloud compute instances list
```

### Create a VM instance

```bash
gcloud compute instances create vm-name
```

### Start a VM

```bash
gcloud compute instances start vm-name
```

### Stop a VM

```bash
gcloud compute instances stop vm-name
```

---

## 5️⃣ Cloud Storage Commands

### List storage buckets

```bash
gcloud storage buckets list
```

### Create a bucket

```bash
gcloud storage buckets create gs://BUCKET_NAME
```

---

## 6️⃣ Kubernetes (GKE) Commands

### List clusters

```bash
gcloud container clusters list
```

### Get cluster credentials

```bash
gcloud container clusters get-credentials CLUSTER_NAME
```

---

## 7️⃣ IAM Commands (Basic)

### List IAM policy bindings

```bash
gcloud projects get-iam-policy PROJECT_ID
```

---

## 8️⃣ Help & Documentation

### Get help for any command

```bash
gcloud help
```

### Help for a specific command

```bash
gcloud compute instances create --help
```

---

## ✅ In Simple Words

* `gcloud` is the CLI to control GCP
* Authentication, configuration, and resource management are core tasks
* These commands form the foundation for **DevOps, SRE, and Cloud Engineer roles**
