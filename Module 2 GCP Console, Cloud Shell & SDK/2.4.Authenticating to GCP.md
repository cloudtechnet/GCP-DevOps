# Authenticating to GCP

In **Google Cloud Platform (GCP)**, **authentication** is the process of verifying **who you are** before accessing cloud resources.

Authentication works together with **IAM (Identity and Access Management)**, which controls **what you are allowed to do** after logging in.

A **Service Account** is a special identity used by applications instead of humans.

### Real-World Example

A backend application running on a VM needs access to Cloud Storage.
Instead of using a user login, it uses a **service account**.
