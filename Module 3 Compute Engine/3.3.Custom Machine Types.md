## Custom Machine Types

In **Google Cloud Platform (GCP)**, **Custom Machine Types** allow you to **define your own virtual machine configuration** by selecting the exact number of **vCPUs and memory** instead of using fixed, predefined VM sizes.

This helps achieve **better performance and cost optimization**.
